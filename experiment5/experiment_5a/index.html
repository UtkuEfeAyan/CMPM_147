Description

For this experiment, I wanted to make a generator that feels like an image is slowly forming over time—not instantly shown, but built through layers of mutated shapes. My idea was to start with large, soft circles and then keep adding smaller, more specific ones as time passed. Basically, create something that slowly becomes more recognizable and detailed. But then I realized the assignment wanted more user control and mutation-based interaction, so I made two versionsThis ones which is more straightforward and fits the assignment exactly (experiment_5a)

Technical

I built a visual mutation system that draws transparent circles onto a canvas to try to match a target image. You can control mutation cycle speed, this is controlled through sliders so the viewer shapes the process directly. There's also a live score tracking (based on color matching error to the target image).
